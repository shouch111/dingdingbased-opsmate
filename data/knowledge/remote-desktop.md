# 笔记本无法远程控制台式机

## 症状
- 使用笔记本电脑远程连接台式机时失败
- 提示无法连接到远程计算机
- 远程桌面或远程控制软件无响应

## 可能原因
- 台式机防火墙禁用了远程连接（推测为火绒安全软件导致）
- 设备未接入同一公司网络

## 解决步骤
1. 解除防火墙对远程连接的限制
2. 联系 IT 卸载台式机上的火绒安全软件
3. 确认笔记本和台式机是否均接入公司网络
4. 两台设备需处于同一网段，可在命令提示符中执行 `ping` 测试连通性
5. 如果以上步骤无效，联系 IT 进一步检查网络策略和远程桌面设置

## 备注
- 具体原因尚未完全确认，推测为火绒安全软件误拦截导致
