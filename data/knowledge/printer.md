# 打印机无法连接

## 症状
- 用户点击打印后无反应
- 打印队列显示"已暂停"或"错误"
- 打印机显示离线状态

## 解决步骤
1. 检查打印机电源和网线是否插好
2. 打开「控制面板」→「设备和打印机」，找到目标打印机
3. 右键点击打印机，选择「查看正在打印的内容」
4. 如果队列中有卡住的任务，全部取消
5. 右键打印机 →「打印机属性」→「端口」→ 确认选中了正确的 IP 端口
6. 如果还不行，重启 Print Spooler 服务：
   - Win+R，输入 services.msc
   - 找到 Print Spooler，右键「重新启动」
7. 如果以上都无效，联系 IT 工程师更换打印机驱动
